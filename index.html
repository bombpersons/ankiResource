ankiResource by bombpersons

A set of web apps written in Django, designed to make a convenient way to share sentences, resources, etc to aid in learning languages.

This is mainly a project written just for me, but if you want to use the code in your own project then feel free =D If you want to see the site running, please go to http://anki-resource.uk.to

Dependencies

libAnki - http://ichi2.net/anki/ Django - http://www.djangoproject.com/

Install

Just clone the repo, and use your prefered method (I used fast_cgi and lighttpd) to deploy the site.
